## Run Locally

Clone the project (or download .zip & extract)

```bash
  git clone https://github.com/diorizqi404/recipe_app.git
```

Go to the project directory

```bash
  cd my-project
```

Get Packages

```bash
  flutter pub get
```

Run the application (make sure emulator is running)

```bash
  flutter run
```
